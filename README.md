# Comparative-Performance-Study-of-XGBoost-and-TCN-in-Customer-Purchase-Prediction
This repository contains the official codebase and research framework developed for my Master's Thesis. The project evaluates and compares the predictive performance of **XGBoost** (Gradient Boosting) and **Temporal Convolutional Networks (TCN)** (Deep Learning) in forecasting customer purchase behaviors.
